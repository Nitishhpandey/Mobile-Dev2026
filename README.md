## 🧠 Key Learnings

### What is React?
React is a JavaScript library used for building user interfaces, especially single-page applications.  
It allows developers to create reusable UI components and manage dynamic data efficiently.

### Why React?
- Component-based architecture
- Faster rendering using Virtual DOM
- Reusable code
- Strong ecosystem and community support

### JSX
JSX allows writing HTML-like syntax inside JavaScript.

Example:
```javascript
function App() {
  return <h1>Hello React</h1>;
}
